¿Qué es un preprocesador de css?

es una forma practica de escribir codigo css, ya que nos permite a partir de una estructura definida crear estilos sin la necesidad de volver a crearlos es como los componentes en react js

Variables

Las variables en scss son de mucha ayuda cuando queremos reutilizar algún color, alguna fuente o algun tamaño y que no sea necesariamente definido en las variables globales

/*Declaración*/ $Nombre:Intrucción;

/*Ejemplo*/ $ColoresPrimarios:blue; H1{ background-color:$ColoresPrimarios; }

Mixin

los Mixin es la forma en la cual nos evitamos poner todo el codigo css en una sola linea de codigo

/*Declaración*/ @mixin nombre{ intrucciones }

/*Ejemplo*/ @mixin flexboxContent{ display:flex; flex-direction:column; } .Contenido{ @include flexboxContent; }

Importaciones

las importaciones nos permiten exportar nuestros estilos para tener una estructura mas organizada

/*Declaración*/ @mixin nombre{ intrucciones } /*Ruta*/

/*Ejemplo*/ @mixin flexboxContent{ display:flex; flex-direction:column; } /*Ruta*/ ../Scss/Datos .Contenido{ @include flexboxContent; }
